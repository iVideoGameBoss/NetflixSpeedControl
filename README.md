# NetflixSpeedControl
Netflix Speed Control is a lightweight Firefox WebExtension that lets you adjust Netflix video playback speeds on the fly. Choose from 1x (normal), 1.5x, 2x, 2.5x, 3x, 3.5x, or 4x via a simple popup UI.
